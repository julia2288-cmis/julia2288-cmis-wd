How to Create a New Web Page

  1. Open the terminal and clone the repository
  2. Type python -m SimpleHTTPServer to start the development server
  3. Type 'gedit (filename).html' to creat a new html
  4. Type !doctypehtml inside of an angle brackets
  5. Type other tags such as head,body, etc that are necessary
  6. Make sure children tags are tabbed
  7. Type 'main.css' on the terminal to add details to the html
  8. After all the details are added, save the files(css&html).
  9. Then type sftp username@cmis-hsteacher for sftp
